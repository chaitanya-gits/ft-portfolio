Project 001 — Web Application — 2025

Digital
Mood Board
Generator

A browser-based mood board tool for fashion ideation. Drag colour swatches onto the canvas, upload reference images, add text notes. Built with vanilla JavaScript—no frameworks or libraries.

Honest note: This is my first project combining web development with fashion thinking. It is simple, but it is fully functional. I built it because physical mood boards were how I first understood how designers like Margiela and Kawakubo conceptualised their collections.
Year
